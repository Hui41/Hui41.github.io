基本概念

设 (M,g) 和 (N,h) 为两个黎曼流形，ϕ:M→N 为一光滑映射。
微分映射 dϕ 可视为 A1(ϕ−1TN)=Γ(T∗(M)⊗ϕ−1TN) 中的元素。
则 dϕ 在 x 点处的范数 |dϕ|x 可定义为线性映射 dϕ|x:(TxM,g)→(Tϕ(x)N,h) 的 Hilbert-Schmidt 范数。

若取 x 和 ϕ(x) 附近的局部坐标为 (xi) 和 (uα)，则

(1)|dϕ|2=gijhαβ(ϕ)ϕiαϕjβ,

其中 ϕiα=∂ϕα∂xi。定义 ϕ 的能量密度为：

(2)e(ϕ):=12|dϕ|2.

定义 ϕ 的能量泛函为：

(1)E(ϕ):=∫Me(ϕ)dvg.

:::

exercise 证明：|dϕ|2=⟨ϕ∗h,g⟩。 :::

:::remark 显然 E(ϕ)≥0，且当且仅当 ϕ 为常数时 E(ϕ)=0。 :::

:::remark 给定 v，可定义一族映射 ϕt，使得 ϕ0=ϕ，且 dϕtdt|t=0=v。
例如取 ϕt(x)=expϕ(x)⁡(tv)，则

(3)DvE(ϕ)=dE(ϕt)dt|t=0.

:::

:::proposition 第一变分公式：

(4)dE(ϕt)dt|t=0=−∫M⟨v,τ(ϕ)⟩dvg,

其中 τ(ϕ):=−d∗dϕ=tr∇dϕ 被称为 ϕ 的张力场。 :::

:::proof 定义 Φ:M×R→N 为 Φ(x,t):=ϕt(x)。那么

(5)ddtE(ϕt)=12∫M∂∂t⟨dϕt,dϕt⟩dvg=∫M⟨∇∂∂tdϕt,dϕt⟩.

其中 ∇∂∂t 为 T∗(M×R)⊗ϕ−1TN 上的共变微分。

对于任意 X∈TM：

(6)(∇∂∂tdϕt)(X)=∇XΦ(∂Φ∂t).

故

(7)(∇∂∂tdϕt)|t=0=∇v.

由于在 A0(ϕ−1TN) 上 ∇ 即为 d，由 Stokes 公式可得：

(8)ddtE(ϕt)|t=0=∫M⟨dv,dϕ⟩dvg=−∫M⟨v,d∗dϕ⟩dvg.

 

definition 称 ϕ 为调和映射若其为能量泛函 E(⋅) 的临界点，即 τ(ϕ)=0。

严格来说，为了使积分 (1) 有限，积分区域应限制在任意有界区域 Ω⊂M。

exercise 在局部坐标下，有：

(9)(∇dϕ)ijα=∂2ϕα∂xi∂xj−Γijkϕkα+Γ~βγαϕiβϕjγ,

以及

(10)τ(ϕ)α=Δϕα+Γ~βγαϕiβϕjγgij.

因此 τ 是一个半线性椭圆方程组算子。

example 若 M=S1，则映射 ϕ:S1→(N,h) 为调和的当且仅当其为弧长参数化的测地线。

example 恒等映射 I:(M,g)→(M,g) 显然是调和的。

example 若 N=R，则调和映射 f:(M,g)→R 就是调和函数。此时也可定义上调和函数、下调和函数。

definition 给定映射 ϕ:(M,g)→(N,h)，称二次型 ∇dϕ 为 ϕ 的第二基本形式。

remark 因为 dϕ∈A1(ϕ−1TN)，所以 M 与 N 的 Levi-Civita 联络诱导了一个 A1(ϕ−1TN) 上的联络 ∇。

proposition 由于 Levi-Civita 联络无挠，∇dϕ 是对称的。

corollary 1-形式 dϕ∈A1(ϕ−1TN) 是闭的。

corollary ϕ 为调和映射当且仅当 dϕ 为调和 1-形式。

example 若 ϕ:(M,g)→(N,h) 为黎曼浸入，则

(11)∇XNY=∇XMY+∇dϕ(X,Y),∀X,Y∈TM.

proposition 一个黎曼浸入为调和的当且仅当它是极小的（minimal）。

复合公式

proposition 给定 M→ϕN→ψP，则

(12)∇d(ψ∘ϕ)=dψ∘∇dϕ+∇dψ(dϕ,dϕ),τ(ψ∘ϕ)=dψ∘τ(ϕ)+tr∇dψ(dϕ,dϕ).

 

注意：一般来说两个调和映射的复合不再是调和映射。

 

proposition 下列说法等价：

  1. ϕ 为全测地的（即 ∇dϕ=0）；

  2. ϕ 为保联络的；

  3. ϕ 将 M 的测地线映为 N 的测地线。

Ishihara 定理

theorem (Ishihara, 1979) 设 ϕ:M→N，则：

  1. ϕ 为全测地的当且仅当其将凸函数芽映为凸函数芽；

  2. ϕ 为调和的当且仅当其将凸函数芽映为次调和函数芽。

remark 一个函数 f:U⊂N→R 称为凸（或严格凸）当且仅当 ∇df≥0（或 >0）。
